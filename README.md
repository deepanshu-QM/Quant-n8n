Quant-n8n is a modern workflow-based quantitative trading platform inspired by automation tools like n8n, designed for traders, developers, and researchers to visually create, automate, test, and execute trading strategies.

The platform combines:

Real-time market data
AI-powered trading agents
Drag-and-drop workflow automation
Multi-condition strategy execution
Modular architecture
Modern full-stack technologies

Built using Bun, TypeScript, React, MongoDB, and React Flow.
